Conectar com Google Sheets

Configure a planilha para atualização automática dos dados

1
Abra o Google Sheets e crie uma planilha nova (ou copie a estrutura abaixo)
2
Crie 3 abas com os nomes exatos: BU · MENSAL · VF
3
Preencha cada aba com os dados (veja a estrutura no botão abaixo)
4
No Google Sheets: Arquivo → Compartilhar → Publicar na web → escolha cada aba → formato CSV → Publicar
5
Cole os 3 links abaixo e clique em Salvar
Aba BU — colunas: BU · Meta Anual · Vendido · Qtd · Ticket · Pipeline · Meta Mes
Ex: PALESTRAS, 1000000, 830000, 2, 415000, 0, 0

Aba MENSAL — colunas: Mes · Palestras · Publicidade · JotaLab · JotaEdition · Advisory · Seleto · Adsense · Total · Meta

Aba VF — colunas: Data · BU · Cliente · Descricao · Valor · Mes · Responsavel
